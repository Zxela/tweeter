# Tweeter Project

Tweeter is a simple, single-page Twitter clone.

## Dependencies

- Express 4.13.4
- NodeJS 5.10.x or above
- Body-parser 1.15.2
-	Chance 1.0.2
- MD5 2.1.0
- MongoDB 2.2.36

## Dev Dependencies

- Nodemon
